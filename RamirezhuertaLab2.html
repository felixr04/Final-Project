How to Define a Table
A table is used to arrange data into rows and columns.
When to Use <tr>
Its used when you want to add a row to your table.
When to Use <th>
This defines a header cell.
When to Use <td>
This marks the begining of a cell or table data.
When to Use <thead>
To group header content in an HTML table.
When to Use <tfoot>
Used to group footer content.
When to Use <caption>
Used to caption a table.
Diffrence between colspan and rowspan
A colspan merges cells in caloumn while a rowspan merges cells in a row
Diffrence between cell spacing and paddling
Cell padding is the space between the cell edges and the cell content while spacing is the space between each cell.
