Terms and Definitions

Doctype Statement
It is an "information" to the browser about what document type to expect.
How to Use <ol>,<ul>, and <dl>
UL is used to represent a list in bullet point or no particular order, OL is used to represent a list in chronological order, DL is used to describe a list of terms with a definition of each term.
Text Editor VS Word Processor
A text editor does not have formatting while a word processor does
Client Server Model
Typically follows a request/response pattern, using a message system to make requests to a server.
FTP
Description:Provides muiltiple access methods for file transfer
Definition:File Transfer Protocol
HTTP
Description:An application layer protocol designed to transfer information between networked devices and runs on top of other layers of the network protocol stack.
Definition:Hypertext Transfer Protocol
TCP/IP
Description:Specifies how data is exchanged over the internet by providing end-to-end communications that identify how it should be broken into packets, addressed, transmitted, routed and received at the destination.
Definition:Transmission Control Protocol
